# Daily Gratitude Tracker

A beautiful Streamlit web app for tracking daily gratitude and mood patterns. Users can log what they're grateful for each day, rate their mood, and visualize their emotional trends over time.

##  Features
- **Daily gratitude logging** with mood tracking
- **Interactive analytics** with mood trends and word clouds
- **Achievement system** and streak tracking
- **Beautiful visualizations** using Plotly and WordCloud

##  Built With
- Python + Streamlit
- Pandas for data handling
- Plotly for interactive charts
- WordCloud for text visualization
